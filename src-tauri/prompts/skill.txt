# Skill Translation Guidelines

## For Skill Names:
1. **Power Clarity**: Convey the skill's power level and effect clearly
2. **Action-Oriented**: Maintain the dynamic, action-focused nature of skill names
3. **Technical Terms**: Preserve magical, technical, or combat terminology accurately
4. **Impact**: Ensure the translation sounds impactful and memorable
5. **System Consistency**: Maintain consistency with the game's combat system
6. **Gameplay Clarity**: Keep descriptions clear for gameplay understanding
7. **Category Context**: Consider if it's an attack, healing, buff, or debuff skill

**Special Considerations for Skill Names:**
- Combat skills: Preserve power level and special effects
- Magic spells: Maintain mystical terminology and effects
- Passive abilities: Keep effect descriptions clear and accurate
- Ultimate skills: Preserve the impressive and powerful nature

**Skill Name Examples:**
- ファイアボール => Fireball
- ヒール => Heal
- 雷撃 => Thunder Strike
- 究極奥義 => Ultimate Technique
- 理性奪取 => Sanity Drain
- 性欲操作 => Libido Manipulation
- 淫魔の誘惑 => Succubus Seduction

**Specific Skill Translations (Use These Exact Translations):**
- **Skill Names:** 攻撃 => Attack, 防御 => Defend, 連続攻撃 => Continuous Assault, ２回攻撃 => Double Attack, ３回攻撃 => Triple Attack, 逃げる => Escape, 様子を見る => Check Up, ヒール => Heal, ファイア => Fire, スパーク => Spark

**Skill Message Guidelines:**
- Translate `の攻撃！` as an action phrase (e.g., "attacks!", "strikes!", "launches an attack!")
- Translate `は身を守っている。` as a defensive phrase (e.g., "is protecting itself", "defends", "guards")
- Translate `は逃げてしまった。` as a fleeing phrase (e.g., "has fled", "escapes", "runs away")
- Translate `は様子を見ている。` as an observation phrase (e.g., "is observing", "watches carefully", "assesses the situation")
- Translate `は%1を唱えた！` as a spell-casting phrase (e.g., "casts %1!", "chants %1!", "uses %1!")
- Maintain the dramatic, action-oriented tone of battle messages
- Preserve the excitement and intensity of combat scenes

## For Skill Descriptions:
1. **Tone Preservation**: Maintain the original tone and style of the description
2. **Formatting**: Preserve any special formatting, punctuation, or line breaks
3. **Formality Level**: Keep the same level of formality as the original
4. **Cultural Adaptation**: Ensure cultural appropriateness for the target audience
5. **Consistency**: Maintain consistency with similar descriptions in the game
6. **Clarity**: Ensure the description remains clear and informative
7. **Combat Details**: Preserve damage, effect, and condition descriptions
8. **Technical Accuracy**: Maintain precise gameplay mechanics and values

**Special Considerations for Skill Descriptions:**
- Combat skill descriptions: Maintain damage and effect descriptions
- Healing skill descriptions: Preserve recovery amounts and conditions
- Buff skill descriptions: Keep enhancement descriptions clear
- Debuff skill descriptions: Maintain negative effect descriptions

**Skill Description Examples:**
- "Deals fire damage to all enemies.\n[Damage: 150%]" => "Deals fire damage to all enemies.\n[Damage: 150%]"
- "Recovers HP for all allies.\n[Recovery: 200 HP]" => "Recovers HP for all allies.\n[Recovery: 200 HP]"
- "Increases attack power for 3 turns.\n[Attack +50%]" => "Increases attack power for 3 turns.\n[Attack +50%]"
- "相手の理性を奪う。\n[Sanity -30]" => "Drains the opponent's sanity.\n[Sanity -30]"
- "性欲を高める効果。\n[Libido +40]" => "Effect that increases libido.\n[Libido +40]"
- "子宮の状態を悪化させる。\n[Womb Status -25]" => "Worsens womb condition.\n[Womb Status -25]"

## For Skill Messages:
1. **Combat Context**: Maintain the dramatic, action-oriented nature of battle messages
2. **Character Voice**: Preserve the speaking character's personality and tone
3. **Emotional Impact**: Keep the excitement and intensity of battle scenes
4. **Formatting**: Preserve any special formatting, punctuation, or line breaks
5. **Consistency**: Maintain consistency with similar battle messages in the game
6. **Clarity**: Ensure the message remains clear and impactful

**Special Considerations for Skill Messages:**
- Battle announcements: Maintain dramatic and exciting tone
- Character dialogue: Preserve character personality and speech patterns
- Effect descriptions: Keep clear and impactful descriptions
- Victory/defeat messages: Maintain appropriate emotional tone

**Skill Message Examples:**
- "Fireball deals massive damage!" => "Fireball deals massive damage!"
- "Healing light restores your strength!" => "Healing light restores your strength!"
- "The enemy is paralyzed!" => "The enemy is paralyzed!"
- "理性を奪われた！" => "Sanity has been drained!"
- "性欲が高まっている！" => "Libido is rising!"
- "子宮の状態が悪化した！" => "Womb condition has worsened!"

## For All Skill Text:
- Maintain combat/magical terminology
- Ensure gameplay clarity and accuracy
- Keep consistent terminology across the game
- Preserve any special characters or codes
- Maintain skill category consistency
