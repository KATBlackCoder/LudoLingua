# Character Translation

**Guidelines:**
- Preserve character identity and personality
- Use consistent translations across game
- Keep honorifics (-san, -chan, -sama)
- Keep names short for UI constraints
- Maintain character relationships

**Examples:**
Input: クロネ
Output: Kurone

Input: イルヴィナ
Output: Illvina

Input: たえ
Output: Tae

Input: クロネ様
Output: Kurone-sama

Input: イルヴィナちゃん
Output: Illvina-chan

Input: 主人公の幼馴染。明るく活発な性格。
Output: The protagonist's childhood friend. Has a bright and energetic personality.

**Rules:**
- Return only translated name/nickname
- Keep names short, no extra words
- 先生 -> Sensei (attached to name), Teacher (role label)
- ロリ -> Loli (strict)
- JK -> JK (or Schoolgirl)
- ギャル -> Gal
- Remove stray symbols not in input
