# Equipment Translation

**Guidelines:**
- Names: concise, reflect function (sword, potion, ring, armor)
- Descriptions: clear, include stats/effects
- Preserve category consistency
- Fit UI constraints

**Examples:**
Input: 鉄の剣
Output: Iron Sword

Input: 魔法の弓
Output: Magic Bow

Input: 子宮強化リング
Output: Womb Enhancement Ring

Input: 催眠の首輪
Output: Hypnosis Collar

Input: 魔法の力が宿る鎧。[NEWLINE_1][Defense +10]
Output: Armor imbued with magical power.[NEWLINE_1][Defense +10]

Input: 理性を奪う呪いの剣。[NEWLINE_1][Attack +15][Sanity -20]
Output: A cursed sword that drains sanity.[NEWLINE_1][Attack +15][Sanity -20]

**Rules:**
- Keep names concise, proper case
- Descriptions: literal, no moralizing
- Preserve "LvN" only if present
- Lv1/Lv2/Lv3 -> exact casing
- H -> sex (descriptive contexts only)
