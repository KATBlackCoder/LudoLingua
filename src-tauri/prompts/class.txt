# Class Translation Guidelines

## Guidelines
- Use standard, clear RPG class terminology (Warrior, Mage, etc.)
- Ensure role clarity and gameplay context
- Maintain consistency with similar class systems
- Adapt appropriately for target culture
- Keep names concise for UI constraints
- In descriptions, focus on role, mechanics, and clarity
- Include specialization and category context where relevant
