# Character Translation Guidelines

## Guidelines
- Preserve character identity and personality
- Ensure readability and naturalness in {target_language}
- Use consistent translations across the game (names, nicknames)
- Preserve honorifics (-san, -chan, -sama, etc.) when present
- Consider role, personality, relationships, and cultural context
- Keep names short for UI constraints
- Maintain consistent character relationships and development

## Examples: Names
Input: クロネ
Output: Kurone

Input: イルヴィナ
Output: Illvina

Input: たえ
Output: Tae

Input: すみれ
Output: Sumire

Input: 主人公
Output: Protagonist

## Examples: Nicknames
Input: クロネ様
Output: Kurone-sama (formal address)

Input: イルヴィナちゃん
Output: Illvina-chan (affectionate)

Input: たえちゃん
Output: Tae-chan (affectionate)

Input: すみれさん
Output: Sumire-san (polite)

## Examples: Profiles
Input: A mysterious girl with silver hair and red eyes.
Output: A mysterious girl with silver hair and red eyes.

Input: 主人公の幼馴染。明るく活発な性格。
Output: The protagonist's childhood friend. Has a bright and energetic personality.

Input: 銀髪赤眼の謎の少女。
Output: A mysterious girl with silver hair and red eyes.

Input: 主人公の幼馴染。明るく活発な性格で、いつも主人公を支えてくれる。
Output: The protagonist's childhood friend. Has a bright and energetic personality, always supporting the protagonist. 

## Output Policy (Character Names)
- Return only the translated name or nickname. No prefixes/suffixes like "Translated:" or quotes unless input had them
- Keep names short; do not add commentary or extra words
- Never include the delimiters (<<TEXT_START>> / <<TEXT_END>>) in the output
- Output must contain only {target_language} glyphs (no CJK)
- Format numeric variants as "Loli {number} …" (add a space after "Loli")
- 先生: use "Sensei" when attached to a name (e.g., Kurone-sensei); use "Teacher" as a role label; never "Mr."
- Do not add punctuation, placeholders, or symbols not present in input

## Terminology Normalization & Banned Terms
- ロリ → Loli (strict). Never use "pedo", "paedo", or similar slurs unless explicitly present in input
- ＪＫ / JK → JK (or Schoolgirl if needed). Do not expand to "Teen", "Teenager"
- ギャル → Gal
- 公園 → Park; 街中 → In Town / Downtown; 駅前 → Station Front; 校庭 → Schoolyard
- Remove stray symbols not in input (e.g., '@')
- Do not invent names not in input
- 先生 → Teacher
