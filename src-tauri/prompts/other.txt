# Other Text Translation Guidelines

## For Note Fields:
1. **Tone Preservation**: Maintain the original tone and style of the text
2. **Formatting**: Preserve any special formatting, punctuation, or line breaks
3. **Formality Level**: Keep the same level of formality as the original
4. **Cultural Adaptation**: Ensure cultural appropriateness for the target audience
5. **Consistency**: Maintain consistency with similar text in the game
6. **Clarity**: Ensure the text remains clear and informative
7. **Context Preservation**: Keep important context and meaning intact

**Note Field Examples:**
Input: Developer comment: This item is rare.
Output: Developer comment: This item is rare.

Input: Hidden treasure location reference.
Output: Hidden treasure location reference.

Input: Special event trigger condition.
Output: Special event trigger condition.

## For Metadata Fields:
1. **Technical Accuracy**: Maintain precise technical information
2. **Formatting**: Preserve any special formatting, codes, or structure
3. **Consistency**: Keep consistent with similar metadata in the game
4. **Clarity**: Ensure technical information remains clear
5. **Reference Preservation**: Keep important references and links intact

**Metadata Field Examples:**
Input: ID: 1001, Type: Weapon
Output: ID: 1001, Type: Weapon

Input: Reference: Map_001, Event_005
Output: Reference: Map_001, Event_005

Input: System: Battle_Trigger_01
Output: System: Battle_Trigger_01
