# Equipment Translation Guidelines

## For Equipment Names:
1. **Function Clarity**: Maintain clear understanding of the equipment's purpose and function
2. **Concise Names**: Keep equipment names short, memorable, and easy to read
3. **Equipment Types**: Preserve specific terminology (sword, potion, ring, armor, etc.)
4. **UI Constraints**: Ensure translations fit within game interface limitations
5. **Category Consistency**: Maintain consistency within equipment categories
6. **Lore Preservation**: Keep important lore and background information
7. **Usage Context**: Consider combat, defensive, enhancement, or consumable properties

**Special Considerations for Equipment Names:**
- **Weapons**: Maintain clear weapon type, material, and combat function
- **Armor**: Preserve defensive properties and material information
- **Accessories**: Keep enhancement and protection properties clear
- **Consumables**: Maintain clear usage and effect information
- **Quest items**: Preserve story significance and purpose
- **Magical equipment**: Keep mystical terminology and effects
- **Legendary equipment**: Preserve unique and powerful nature

**Equipment Name Examples:**
- **Weapons:** 鉄の剣 => Iron Sword, 魔法の弓 => Magic Bow, 聖なる剣 => Sacred Sword
- **Armor:** 鉄の盾 => Iron Shield, 魔法の鎧 => Magic Armor, 聖なる兜 => Sacred Helm
- **Accessories:** 魔法の指輪 => Magic Ring, 幸運のチャーム => Lucky Charm
- **Consumables:** 回復薬 => Healing Potion, 媚薬 => Aphrodisiac, 理性回復薬 => Sanity Recovery Potion
- **Magical:** 理性奪取の剣 => Sanity Drain Sword, 性欲操作の杖 => Libido Manipulation Staff

**Specific Equipment Translations (Use These Exact Translations):**
- **Items:** ポーション => Potion, マジックウォーター => Magic Water, ディスペルハーブ => Dispel Herb, スティミュラント => Stimulator
- **Weapons:** 剣 => Sword, 斧 => Axe, 杖 => Staff, 弓 => Bow
- **Armor:** 盾 => Shield, 帽子 => Hat, 服 => Clothes, 指輪 => Ring

## For Equipment Descriptions:
1. **Tone Preservation**: Maintain the original tone and style of the description
2. **Formatting**: Preserve any special formatting, punctuation, or line breaks
3. **Formality Level**: Keep the same level of formality as the original
4. **Cultural Adaptation**: Ensure cultural appropriateness for the target audience
5. **Consistency**: Maintain consistency with similar descriptions in the game
6. **Clarity**: Ensure the description remains clear and informative
7. **Property Details**: Preserve combat stats, defensive values, enhancement effects, or usage instructions
8. **Lore Elements**: Keep story significance and background information

**Special Considerations for Equipment Descriptions:**
- **Weapon descriptions**: Preserve damage, accuracy, and special combat properties
- **Armor descriptions**: Maintain defensive values and protection details
- **Accessory descriptions**: Keep enhancement and protection details
- **Consumable descriptions**: Preserve usage instructions and effect details
- **Quest item descriptions**: Maintain story significance and usage context
- **Magical equipment**: Preserve mystical effects and enchantment details
- **Legendary equipment**: Maintain impressive and powerful descriptions

**Equipment Description Examples:**
- **Weapons:** "A sturdy iron sword.\n[Attack +5]" => "A sturdy iron sword.\n[Attack +5]"
- **Armor:** "魔法の力が宿る鎧。\n[Defense +10]" => "Armor imbued with magical power.\n[Defense +10]"
- **Accessories:** "A cute pink heart charm.\nA girly accessory that adds a feminine touch." => "A cute pink heart charm.\nA girly accessory that adds a feminine touch."
- **Consumables:** "Restores HP when consumed.\n[Recovery: 100 HP]" => "Restores HP when consumed.\n[Recovery: 100 HP]"
- **Magical:** "理性を奪う呪いの剣。\n[Attack +15][Sanity -20]" => "A cursed sword that drains sanity.\n[Attack +15][Sanity -20]"
- **Erotic:** "性欲を高める媚薬。\n[Libido +50]" => "An aphrodisiac that increases libido.\n[Libido +50]"

## For All Equipment Text:
- Maintain fantasy/medieval terminology
- Ensure gameplay clarity and accuracy
- Keep consistent terminology across the game
- Preserve any special characters or codes
- Maintain equipment category consistency
- Consider combat, defensive, enhancement, and consumable contexts
