# Equipment Translation Guidelines

## Guidelines
- Keep names concise and clear; reflect function/type (sword, potion, ring, armor)
- Preserve category consistency (Weapons/Armor/Accessories/Consumables)
- Ensure descriptions are clear and useful; include stats/effects where present
- Fit UI constraints and maintain lore/style consistency

## Examples: Equipment Names
Input: 鉄の剣
Output: Iron Sword

Input: 魔法の弓
Output: Magic Bow

Input: 聖なる剣
Output: Sacred Sword

Input: 子宮強化リング
Output: Womb Enhancement Ring

Input: 催眠の首輪
Output: Hypnosis Collar

Input: 貞操帯
Output: Chastity Belt

## Examples: Descriptions
Input: A sturdy iron sword.[NEWLINE_1][Attack +5]
Output: A sturdy iron sword.[NEWLINE_1][Attack +5]

Input: 魔法の力が宿る鎧。[NEWLINE_1][Defense +10]
Output: Armor imbued with magical power.[NEWLINE_1][Defense +10]

Input: 理性を奪う呪いの剣。[NEWLINE_1][Attack +15][Sanity -20]
Output: A cursed sword that drains sanity.[NEWLINE_1][Attack +15][Sanity -20]

Input: 性欲を高める媚薬。[NEWLINE_1][Libido +50]
Output: An aphrodisiac that increases libido.[NEWLINE_1][Libido +50]

Input: 子宮の状態を改善する薬。[NEWLINE_1][Womb Status +30]
Output: Medicine that improves womb condition.[NEWLINE_1][Womb Status +30]

## Examples: Categories
- Weapons: Swords, bows, staffs, daggers
- Armor: Helmets, chest pieces, gauntlets, boots
- Accessories: Rings, necklaces, charms
- Consumables: Potions, scrolls, food
- Special: Quest items, key items

## Output Policy (Equipment Names/Descriptions)
- Names: concise, proper case (e.g., Lv1 Hypnosis App). Do not add slang or commentary
- Descriptions: clear and literal; do not add moralizing or invented mechanics
- Do not invent levels or categories; preserve "LvN" only if present in input
- If name is slang/brand and unclear, transliterate to English phonetics (katakana -> romaji-style)
- Descriptions: prefer neutral capability phrasing ("Enables …", "Allows … when possessed"); avoid imperatives
- Never introduce placeholder tokens (e.g., [NEWLINE_1]) unless present in input

## Normalization
- Lv1 / Lv2 / Lv3 -> preserve exact casing and spacing ("Lv1", not "LV1" or "lv 1")
- H -> sex (only in descriptive contexts; keep single letter H if input is exactly "H")
